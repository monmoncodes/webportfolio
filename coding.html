Coding as an aid for digital transformation in architecture

Projects

Alpha Coaster.gh

Aided with grasshopper software, this script is designed to understand how many rides are efficient to maintain the operations in the long run. This also calculates how much time it takes for rides to go up and down, therefore understanding the feasibility side of things

Sketch2mass.gh

This grasshopper script allows freehand sketch to be translated to 3d model. this plugin helps out senior architects translate their drafts faster.

Testfit prototyping

This project is designed to tailor building codes to the site context and maximize design solutions.. This will help out when computing building efficiency and budget costing. This also helps out to give the best solution based on clients requests e.g. leasable vs non leasable space. how tall buildings can go. how many sunlight are we allowing in the building and so much more

Storefront Visiblility Analysis

The script is developed to analyze the visibility of storefront when designing shopping malls. This could aid in making business development understand where the prime spots are. Diagrams can be useful to also understand which spaces have blindspots
